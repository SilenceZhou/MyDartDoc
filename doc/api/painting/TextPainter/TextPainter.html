TextPainter constructor

TextPainter({TextSpan text, TextAlign textAlign: TextAlign.start, TextDirection textDirection, double textScaleFactor: 1.0, int maxLines, String ellipsis, Locale locale, StrutStyle strutStyle })

Creates a text painter that paints the given text.

The text and textDirection arguments are optional but text and textDirection must be non-null before calling layout.

The textAlign property must not be null.

The maxLines property, if non-null, must be greater than zero.
