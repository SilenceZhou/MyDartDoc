recognizer property

GestureRecognizer recognizer
final

A gesture recognizer that will receive events that hit this text span.

TextSpan itself does not implement hit testing or event dispatch. The object that manages the TextSpan painting is also responsible for dispatching events. In the rendering library, that is the RenderParagraph object, which corresponds to the RichText widget in the widgets layer; these objects do not bubble events in TextSpans, so a recognizer is only effective for events that directly hit the text of that TextSpan, not any of its children.

TextSpan also does not manage the lifetime of the gesture recognizer. The code that owns the GestureRecognizer object must call GestureRecognizer.dispose when the TextSpan object is no longer used.
