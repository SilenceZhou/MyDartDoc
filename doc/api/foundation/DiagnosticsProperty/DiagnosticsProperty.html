DiagnosticsProperty<T> constructor

DiagnosticsProperty<T>(String name, T value, { String description, String ifNull, String ifEmpty, bool showName: true, bool showSeparator: true, Object defaultValue: kNoDefaultValue, String tooltip, bool missingIfNull: false, DiagnosticsTreeStyle style: DiagnosticsTreeStyle.singleLine, DiagnosticLevel level: DiagnosticLevel.info })

Create a diagnostics property.

The showName, showSeparator, style, missingIfNull, and level arguments must not be null.

The level argument is just a suggestion and can be overridden if something else about the property causes it to have a lower or higher level. For example, if the property value is null and missingIfNull is true, level is raised to DiagnosticLevel.warning.
