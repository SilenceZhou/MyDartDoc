Uint8List getUint8List (int length)

Reads the given number of Uint8s from the buffer.

Implementation

Uint8List getUint8List(int length) {
  final Uint8List list = data.buffer.asUint8List(data.offsetInBytes + _position, length);
  _position += length;
  return list;
}
