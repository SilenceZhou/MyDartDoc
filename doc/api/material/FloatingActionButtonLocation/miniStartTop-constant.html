miniStartTop constant

FloatingActionButtonLocation const miniStartTop = const _MiniStartTopFloatingActionButtonLocation()

Start-aligned FloatingActionButton, floating over the transition between the Scaffold.appBar and the Scaffold.body, optimized for mini floating action buttons.

This is intended to be used with FloatingActionButton.mini set to true, so that the floating action button appears to align with CircleAvatars in the ListTile.leading slot of a ListTile in a ListView in the Scaffold.body.

This is unlikely to be a useful location for apps that lack a top AppBar or that use a SliverAppBar in the scaffold body itself.
