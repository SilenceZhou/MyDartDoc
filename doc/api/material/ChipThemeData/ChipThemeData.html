ChipThemeData constructor

const ChipThemeData({@required Color backgroundColor, Color deleteIconColor, @required Color disabledColor, @required Color selectedColor, @required Color secondarySelectedColor, @required EdgeInsetsGeometry labelPadding, @required EdgeInsetsGeometry padding, @required ShapeBorder shape, @required TextStyle labelStyle, @required TextStyle secondaryLabelStyle, @required Brightness brightness, double elevation, double pressElevation })

Create a ChipThemeData given a set of exact values. All the values must be specified except for elevation and pressElevation, which may be null.

This will rarely be used directly. It is used by lerp to create intermediate themes based on two themes.
