decoration property

Decoration decoration
final

The decoration to paint on the nearest ancestor Material widget.

A shorthand for specifying just a solid color is available in the constructor: set the color argument instead of the decoration argument.

A shorthand for specifying just an image is also available using the new Ink.image constructor.
