margin property

EdgeInsetsGeometry margin
final

The empty space that surrounds the card.

Defines the card's outer Container.margin.

If this property is null then ThemeData.cardTheme.margin is used, if that's null, the default margin is 4.0 logical pixels on all sides: EdgeInsets.all(4.0).
