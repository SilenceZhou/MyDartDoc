FilterChip constructor

const FilterChip({Key key, Widget avatar, @required Widget label, TextStyle labelStyle, EdgeInsetsGeometry labelPadding, bool selected: false, @required ValueChanged<bool> onSelected, double pressElevation, Color disabledColor, Color selectedColor, String tooltip, ShapeBorder shape, Clip clipBehavior: Clip.none, Color backgroundColor, EdgeInsetsGeometry padding, MaterialTapTargetSize materialTapTargetSize, double elevation, ShapeBorder avatarBorder: const CircleBorder() })

Create a chip that acts like a checkbox.

The selected, label, and clipBehavior arguments must not be null. The pressElevation and elevation must be null or non-negative. Typically, pressElevation is greater than elevation.
