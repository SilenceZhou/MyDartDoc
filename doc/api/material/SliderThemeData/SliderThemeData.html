const SliderThemeData({@required double trackHeight, @required Color activeTrackColor, @required Color inactiveTrackColor, @required Color disabledActiveTrackColor, @required Color disabledInactiveTrackColor, @required Color activeTickMarkColor, @required Color inactiveTickMarkColor, @required Color disabledActiveTickMarkColor, @required Color disabledInactiveTickMarkColor, @required Color thumbColor, @required Color disabledThumbColor, @required Color overlayColor, @required Color valueIndicatorColor, @required SliderTrackShape trackShape, @required SliderTickMarkShape tickMarkShape, @required SliderComponentShape thumbShape, @required SliderComponentShape overlayShape, @required SliderComponentShape valueIndicatorShape, @required ShowValueIndicator showValueIndicator, @required TextStyle valueIndicatorTextStyle })

Create a SliderThemeData given a set of exact values. All the values must be specified.

This will rarely be used directly. It is used by lerp to create intermediate themes based on two themes.

The simplest way to create a SliderThemeData is to use copyWith on the one you get from SliderTheme.of, or create an entirely new one with SliderThemeData.fromPrimaryColors.
