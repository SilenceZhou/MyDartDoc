buildTransitions<T> method

Widget buildTransitions <T>(PageRoute<T> route, BuildContext context, Animation<double> animation, Animation<double> secondaryAnimation, Widget child)

Wraps the child with one or more transition widgets which define how route arrives on and leaves the screen.

The MaterialPageRoute.buildTransitions method looks up the current current PageTransitionsTheme with Theme.of(context).pageTransitionsTheme and delegates to this method with a PageTransitionsBuilder based on the theme's ThemeData.platform.
