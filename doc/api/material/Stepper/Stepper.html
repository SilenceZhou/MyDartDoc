Stepper constructor

Stepper({Key key, @required List<Step> steps, ScrollPhysics physics, StepperType type: StepperType.vertical, int currentStep: 0, ValueChanged<int> onStepTapped, VoidCallback onStepContinue, VoidCallback onStepCancel, ControlsWidgetBuilder controlsBuilder })

Creates a stepper from a list of steps.

This widget is not meant to be rebuilt with a different list of steps unless a key is provided in order to distinguish the old stepper from the new one.

The steps, type, and currentStep arguments must not be null.
