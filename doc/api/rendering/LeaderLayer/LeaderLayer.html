LeaderLayer constructor

LeaderLayer({@required LayerLink link, Offset offset: Offset.zero })

Creates a leader layer.

The link property must not be null, and must not have been provided to any other LeaderLayer layers that are attached to the layer tree at the same time.

The offset property must be non-null before the compositing phase of the pipeline.
