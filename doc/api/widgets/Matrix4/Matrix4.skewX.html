Matrix4.skewX(double alpha)

Skew matrix around X axis

Implementation

factory Matrix4.skewX(double alpha) {
  final Matrix4 m = new Matrix4.identity();
  m._m4storage[4] = math.tan(alpha);
  return m;
}
