void updateKeepAlive ()

Ensures that any AutomaticKeepAlive ancestors are in a good state, by firing a KeepAliveNotification or triggering the KeepAliveHandle as appropriate.

Implementation

@protected
void updateKeepAlive() {
  if (wantKeepAlive) {
    if (_keepAliveHandle == null)
      _ensureKeepAlive();
  } else {
    if (_keepAliveHandle != null)
      _releaseKeepAlive();
  }
}
