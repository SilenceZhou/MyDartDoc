Viewport constructor

Viewport({Key key, AxisDirection axisDirection: AxisDirection.down, AxisDirection crossAxisDirection, double anchor: 0.0, @required ViewportOffset offset, Key center, double cacheExtent, List<Widget> slivers: const [] })

Creates a widget that is bigger on the inside.

The viewport listens to the offset, which means you do not need to rebuild this widget when the offset changes.

The offset argument must not be null.
