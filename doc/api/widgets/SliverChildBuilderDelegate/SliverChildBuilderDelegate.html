const SliverChildBuilderDelegate(IndexedWidgetBuilder builder, { int childCount, bool addAutomaticKeepAlives: true, bool addRepaintBoundaries: true, bool addSemanticIndexes: true, SemanticIndexCallback semanticIndexCallback: _kDefaultSemanticIndexCallback, int semanticIndexOffset: 0 })

Creates a delegate that supplies children for slivers using the given builder callback.

The builder, addAutomaticKeepAlives, addRepaintBoundaries, addSemanticIndexes, and semanticIndexCallback arguments must not be null.
